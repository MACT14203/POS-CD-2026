# Órdenes de Café (GitHub Pages)

## Publicar
1. Crea un repo (ej: `ordenes-cafe`)
2. Sube TODO este proyecto a la raíz del repo
3. Settings → Pages → Branch: `main` / Folder: `/root`
4. Abre el enlace que te genera GitHub Pages
